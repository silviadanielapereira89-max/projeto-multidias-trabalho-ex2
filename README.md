# Projeto Multidias Trabalho

## Descrição
Projeto de simulação de trabalho multi-dias utilizando GitHub Desktop.

## Estrutura
- **documentos/**: Ficheiros de planeamento e documentação
- **reunioes/**: Atas de reuniões
- **notas/**: Anotações diárias

## Dias de Trabalho
- Dia 1: Organização inicial
- Dia 2: Atualizações e novos documentos
- Dia 3: Consolidação final
